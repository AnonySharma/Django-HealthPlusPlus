ABOUT US

Health++ is an application that helps user to manage their diet and calories consumption. It provides a common platform for dieticians and their patients, so that they can manage a proper diet.

This project is still in development phase, but when it will be released one can be able to use these features:
  • Visit profiles of other users
  • Show more statistics on the profile page
